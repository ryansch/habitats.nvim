# habitats.nvim
"Habitats" for your neovim projects

## Install

Add to packer config:

```lua
  use {
    "ryansch/habitats.nvim",
    requires = {
      "nvim-lua/plenary.nvim",
      "nvim-telescope/telescope-file-browser.nvim",
      "natecraddock/sessions.nvim",
      "natecraddock/workspaces.nvim"
    }
  }
```
